生物实验拉丁词根简称

发表于 2025-03-20 本文字数： 235 阅读时长 ≈ 1 分钟

In vivo

在活体中的，实验在完整的活生物体中进行。

In vitro

在玻璃中的，实验在生物体外，通常是实验玻璃器皿中进行。这种实验的优点是可以被精确的控制，缺点是实验状态不能很好的反映真实的生物体状态。

Ex vivo

活体外的，其定义是“它是指在生物体以外的人工环境中，以最小自然条件的变化在活组织内或活组织上进行的实验或测量”。

In silico

硅上的，意为通过计算模拟所进行的实验。

In situ

在原位，意指当前状态下，包括该时间点，该拓扑结构下，该测序环境下的泛指。因此，in situ spatio-temporal指的是该时空组学环境下的状态，in situ sequence指的是对应的测序环境下。

北邮国院毕设早期要点总结（Specification规范指导30min视频总结）

发表于 2025-01-13 更新于 2025-01-14 本文字数： 548 阅读时长 ≈ 2 分钟

均为本人自整理要点，或有纰漏，以学院手册为准。

1.规范撰写

500-800词或1-2页，提供的内容框架仅供参考，可删减也可另加其它部分，主要有：

– An initial analysis of user requirements, and how data will be collected

– The algorithms, methodologies and other techniques to be employed

– An initial specification of how users will interact with the system (implementation)

– Experiments that should be done to prove the project hypotheses (research)

– Programming language / database/ software package and hardware to be used

– A list of background material consulted including webpages (Use the Vancouver referencing style)

2.道德规范

必须与导师确认毕设是否包含人类/动物参与，若参与需要填写手册附加页内12个问题；若无参与，需要与导师讨论作弊、AI伦理等的道德风险（AI方向毕设必须包括AI伦理的讨论），详见手册4.1。

若无风险，直接写“没有道德问题”。

3.中期目标

必须是有实物的东西。如正常工作的功能、数据的收集、模型的搭建、训练、仿真设计等这将作为中期评估的标准。

4.甘特图

把导师给的四个任务复制到大任务标题，然后自行拆分小任务，可以不是4个，如3、5、6，直接在word加就可以。

5.敏感主题

所有毕设相关文档不能和军事相关，包括军事用途、参考文献涉军、部分军民两用的技术等。如谨慎讨论雷达、网络安全。

6.导师

导师将在提交后同意/拒绝，被拒绝的有一次时间在12月的修改机会，若继续拒绝，本次毕设将挂科。

7.例子

介绍视频14：30至最后为以往毕设规范文档展示，可以在写作时参考其行文技巧。

8.时间线

  1. 早期报告5%在1.10（本学期期末结束），形式是文档；
  2. 中期报告5%在3.3-7（下学期开学第二周），形式是汇报；
  3. 晚期报告5%在4.9-17，形式是文档+模拟答辩；
  4. 文档提交30%在4月底；最终答辩55%在5.5-19，形式是不会透露导师信息的盲审答辩。
